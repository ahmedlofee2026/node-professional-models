# 🚀 Node Database Explorer - VS Code Extension

متصفح قواعد البيانات الاحترافي لـ Node.js - Professional MongoDB/Database Explorer

## ✨ الميزات

✅ **اتصال سهل بـ MongoDB** - اتصل بقاعدة بيانات MongoDB مباشرة من VS Code  
✅ **عرض المجموعات** - تصفح جميع مجموعاتك في واجهة بسيطة  
✅ **عرض المستندات** - عرض المستندات في جداول سهلة القراءة  
✅ **إدارة المستندات** - أضف وعدّل وحذف المستندات مباشرة  
✅ **البحث المتقدم** - ابحث عن المستندات باستخدام استعلامات MongoDB  
✅ **التصدير والاستيراد** - صدّر واستورد البيانات بصيغة JSON  
✅ **الإحصائيات** - عرض إحصائيات قاعدة البيانات والمجموعات  
✅ **إنشاء مجموعات** - أنشئ مجموعات جديدة مباشرة من الإضافة  
✅ **واجهة سهلة** - تصميم بديهي وسهل الاستخدام  
✅ **اختصارات لوحة المفاتيح** - اختصارات سريعة للعمليات الشائعة  

---

## 🔧 التثبيت

### من VS Code Marketplace
1. افتح VS Code
2. اذهب إلى Extensions (Ctrl+Shift+X)
3. ابحث عن "Node Database Explorer"
4. اضغط Install

### من الملفات
```bash
git clone https://github.com/ahmedlofee2026/node-database-explorer.git
cd node-database-explorer/vscode-extension
npm install
npm run compile
